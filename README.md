
# Clinic Management System

A full-stack **Clinic Management System** developed using **Java, JSP, Servlets, JDBC, and MySQL** to manage clinic operations such as patient records, billing, inventory, and authentication.

##  Features

* Secure user authentication
* Patient management
* Billing and invoice generation
* Inventory management
* Role-based access control

##  Tech Stack

**Backend:** Java, JSP, Servlet, JDBC
**Database:** MySQL
**Frontend:** HTML, CSS, JavaScript
**Tools & Deployment:** Apache Tomcat, Docker, Render, Git, GitHub

##  Project Highlights

* Designed using MVC-based architecture principles
* Integrated database-driven CRUD operations
* Implemented secure authentication and access control
* Deployed on cloud for live access
